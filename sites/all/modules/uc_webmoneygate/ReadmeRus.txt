Модуль для оплаты товаров для сайтов на CMS Drupal + Ubercart с помощью WebMoney merchant interface.

Для работы с данным модулем у Вас должен быть аттестат продавца.

Все действия связанные с получением аттестата можно узнать на сайте http://merchant.webmoney.ru 

После включения модуля мы заходим на - вашсайт/admin/store/settings/payment/edit/methods
И проставляем следующие значения:
1. Имя магазина - название Вашего магазина который вы указали в http://merchant.webmoney.ru
2. WMID - собственно Ваш WMID
3. Секретный ключ - пароль который Вы указывали при регистрации магазина на мерчанте (необходим только при работе по HTTPS как параметр для безопасности).
4. Детали платежа - описание деталей платежа (к примеру оплата за товар, в текущей версии писать латиницей).
5. Вводите номера кошельков на которые вы будете принимать платежи (вводить только те кошельки для которых прописаны настройки в мерчанте).
6. Указываете курсы валют относительно базовой валюты сайта (пример: базовая валюта гривна, для кошелька WMU выставляем курс 1, для доллара 0.12 и т.д.).
7. Выбираете статус который будет присвоен ордеру при успешной оплате (в след версиях модуля будет упразднено).
8. Выбираете режим работы модуля (по умолчанию включен тестовый - все платежи проходят).
!!! В ТЕСТОВОМ РЕЖИМЕ ПЕРЕВОД СРЕДСТВ НЕ ОСУЩЕСТВЛЯЕТСЯ !!!
9. Задаете текст для кнопки оплаты.
10. Задаете текст для страницы ошибочного платежа.

Указывать в мерчанте:
Result URL: ваш_сайт/cart/webmoney/result наличие галочки на предварительный запрос не важно, но для повышения безопасности рекомендуется включить.
Success URL: ваш_сайт/cart/webmoney/success
Fail URL: ваш_сайт/cart/webmoney/fail
метод передачи данных в Success URL и Fail URL POST.

остальные настройки согласно требованиям http://merchant.webmoney.ru
