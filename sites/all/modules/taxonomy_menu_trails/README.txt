Taxonomy Menu Trails.

Quick start
-----------

Configure module for node types and corresponding taxonomy vocabularies you want
to expand the menu:
- Go to Administer >> Site configuration >> Taxonomy Menu Trails settings page
- Enable vacabularies for each node type
- Save settings
